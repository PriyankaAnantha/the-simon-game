Game Description

In this game you need to follow the sequence of colors and repeat them.

Each level gets progressively harder.

Can you keep up?



How to Play

  1. Press any key to start the game.

  2. Watch the sequence of colors.

  3. Repeat the sequence by clicking the buttons in the correct order.

  4. If you make a mistake, the game will restart.

  5. While playing the game, if you hit any keys or the "Start Game" button, the game will restart! Be attentive!

  6. Try to reach the highest level possible!

    And don't forget to contemplate the zenith of your memory powers!
